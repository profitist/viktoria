# FEAT-0001-ISSUE-004: Решено

## Что исправлено
`useEffect` переведён на пустой массив зависимостей `[]` — запускается только при монтировании. Для безопасного доступа к актуальным версиям `logout` и `refresh` внутри эффекта используются ref-обёртки (`logoutRef`, `refreshRef`), которые обновляются при каждом рендере. Это устраняет перезапуск эффекта при каждой навигации (когда менялись identity функций).
